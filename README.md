# Python Calculator version1

This is a basic command-line calculator built with Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division with zero handling
- Input validation

## How to Run
1. Install Python
2. Open your terminal or command prompt
3. Run the file using:
   python calculator.py
